# ReadMe
## <Web 개발에 사용하는 프론트단의 기초 정보기술을 정리했습니다.>
- HTML5
- CSS3 
- Javascript
- Jquery
- 여러 Animation 기법 , 레이아웃
- 페이징 , 이미지 슬라이드 등등 다양합니다.

프로젝트를 진행할때 복습하기 좋게 주석도 첨부 하였고 시행 착오도 버전별러 정리 되어있으며 손코딩 되어 있어 가져다 사용하기도 좋습니다.
레이아웃이나 간단한 기능을 활용해 웹페이지를 꾸며 보세요
